### Pokémon API Server

This directory contains the backend server for the Pokémon Explorer application.

#### Technologies Used:

- Node.js
- Express
- TypeScript
- GraphQL

#### To Run just the API Server:

1. Install dependencies:

   ```bash
   npm install
   ```

2. Build the TypeScript code:

   ```bash
   npm run build
   ```

3. Start the server:
   ```bash
   npm run serve
   ```
