### Pokémon Explorer UI

This directory contains the frontend UI for the Pokémon Explorer application.

#### Technologies Used:

- Vite
- React
- TypeScript
- Apollo Client
- Tailwind CSS
- React Router

#### To Run just the UI:

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
    npm run dev
   ```
